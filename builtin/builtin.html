This page is managed with built-in transitions. To use those, put them in the options:

$stateProvider.state("builtin", { url: "/builtin", templateUrl: 'builtin/builtin.html', controller: 'BuiltinCtrl', lumx: { enter: { options: { duration: 500, easing: 'easeInOutCirc', builtin: 'fromLeft fadeIn' } }, leave: { options: { duration: 500, easing: 'easeInOutCirc', builtin: 'toLeft fadeOut' } } } });
