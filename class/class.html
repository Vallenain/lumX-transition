This page is managed with classes. lumx-transition make some classes available, but you can make your own. To use classes, put them in the options:

$stateProvider.state("class", { url: "/class", templateUrl: 'class/class.html', controller: 'ClassCtrl', lumx: { enter: { options: { duration: 500, easing: 'easeInOutCirc', classes: 'lumx-transition--scale-up' } }, leave: { options: { duration: 500, easing: 'easeInOutCirc', classes: 'lumx-transition--scale-down' } } } });

Remember the order of the classes is important, and the style you might have applied directly to the element will prevail.
